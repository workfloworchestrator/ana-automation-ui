Network Service Interface management and monitoring tools

AuRA

Automated Resource Allocation - the NSI user provider agent for managing network reservations across the ANA network.

Safnari

NSI Connection Service aggregator - manages and monitors NSI connections across multiple network domains.

DDS

Document Distribution Service - discovers and distributes NSI topology documents and network descriptions.

REST API proxy for the DDS - provides a simplified interface to query topologies, STPs, and SDPs.
